Sometimes you want to highlight a link and so you style css to make it an attractive button.

You have a few options:

  1. Make an image in photoshop and use alt text
  2. Make a link and then use a css image as the background

Centreing text in the button vertically can be difficult with option 2.

The solution that seems to work is at:

  • http://www.jakpsatweb.cz/css/css-vertical-center-solution.html
