Jquery has a scroll event which fires when a user updates the scroll position of an element.

To keep a floating menu on screen:

Make its position relative/absolute

$(window) creates a jquery instance of the window object.

get the $(document).scrollTop() value and set the top of the menu 's css to this value.

For smoother scrolling use animate and use stop as scroll fires alot.
$(window).scroll(function() { $('#navigation').stop() .animate({top: $(document).scrollTop()},'slow','easeOutBack');});

I have actually done a nice floating menu with classes.

The scrollTo plugin is also quite handy.
