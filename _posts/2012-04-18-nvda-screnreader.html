NVDA Screenreader

  • Insert = nvda key
  • Browse Mode and Focus Mode
  • Switch by pressing nvda+space
  • Enter or space will switch to control mode on controls
  • Esc will switch back


Browse Mode

  • Cursor keys
  • H:heading
  • 1..6:headings
  • L:list
  • T:talk
  • D:aria landmark

Focus Mode

Elements List

Nvda + f7

Nvda + tab – current focus

Shift – pause speech
