There are webapps which are like webpages and run in webkit. You can run a webapp offline.

Then there are native apps which you get through the appstore. They have access to more native features of the device.

You can use phonegap to compile html, css, and javascript to a native app but this requires xcode.

Xcode requires a developer license or osx lion to download.
