General

  • Cmd-F5 turn on/off
  • vo = voiceover keys = ctrl-opt
  • vo-k keyboard help
  • vo-h help
  • vo-space select
  • vo-cmd voice settings
  • vo-shift down arrow enter element
  • vo-shift up arrow exit element
  • within elements use vo and arrow keys

Tables

  • vo-c-c column
  • vo-r row
  • vo-c header

OS

  • vo-d dock
  • vo-m menu

Webpages

  • vo-u rotor
  • vo-cmd-n, vo-cmd-shift-n auto web spots
