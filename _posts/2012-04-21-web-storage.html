Local storage

  • Store data for your entire website permanently


Session storage

  • Store data for your website for a tab or single window. Expires when tab/window closed but remains if user travels to another website.
  • Both are linked to your domain
    ie. available on all pages on your domain
  • Browser specific

Set

localStorage["user_name"]="Chris Chudleigh";

Get

var x=localStorage["user_name"];

Example:

if (localStorage["user_name"] == null) { alert ("You haven't entered a name yet.");}
else {// Put the name into a text box
. document.getElementById("userName").value = localStorage["user_name"];}
