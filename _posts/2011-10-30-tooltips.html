Browser tooltips appear when using the title attribute in browsers.

Usually you style the text with a dotted underline to indicate a tooltip and change the cursor to a help icon.

An info icon can be used as well.

Problems:
These cannot be styled with CSS
These can only contain text

Javascript can be used to grab the content of the title tag and place it in a generated div tag which can then be styled and then mouseovers to show and hide. You still cannot use rich text tho as the text is grabbed from the title tag.

To do this you need to use an anchor tag is also used with a span tag inside which is hidden until the a tag is hovered over. Not sure if this works in safari. This may contain html. This method can be done with css only.

Javascript can be used to add delay effects and to use ajax.

The main problem with CSS only tooltips is they add extra code to the html which does not make sense for search engines or screenreaders. Probably better off using javascript tooltips as in progressive enhancement.
